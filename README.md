# にんげんタワーバトル（2018年作成）

- 有名なスマートフォンアプリどうぶつタワーバトルを応用したもので，kinectで取得した人間の姿勢をunityちゃんに反映させ，それを交互に土台に積み上げていき，落とした方が負けのゲーム

- kinect2への接続が必須，wiiリモコンをPCにつなげればwiiリモコンでも操作ができる
